# Budget Manager v1.0

Aplikacja desktopowa (C# WinForms) do zarządzania budżetem domowym, stworzona w ramach projektu zaliczeniowego.

##Autorzy

- **Hubert Rudowicz** (Lider, Logika biznesowa, baza danych)
- **Piotr Dobry** (Logowanie, rejestracja)
- **Piotr Janas** (Interfejs użytowania, funkcjonalności)

##Funkcjonalności

Aplikacja umożliwia pełną kontrolę nad finansami:

- **Dashboard** Podgląd stanu konta w określonym przedziale czasu i lista wydatków/przychodów
- **Transakcja** Dodawanie, edycja, usuwanie, filtracja wydatków oraz przychodów (CRUD)
- **Cele oszczędnościowe** Pasek postępu wizualizujący oszczędności na konkretny cel.
- **Export/Import** Możliwość tworzenia kopii zapasowej danych do pliku JSON
- **Raport/Wykresy** Wykresy graficzne wydatków/przychodów oraz kategoryzacji wydatków w wybranym przedziale czasu

##Technologie

- C# .NET 8.0
- Windows Forms (WinForms)
- System.Text.Json (Serializacja danych)

##Jak uruchomić

1. Pobierz repozytorium
2. Otwórz plik `BudgrzeManager.sln` w Visual Studio 2022
3. Zbuduj projekt i uruchom (F5)
